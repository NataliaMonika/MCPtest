# MCP Test — ButtonSolid (Emotion)

Pliki wygenerowane z Figma (komponent `ButtonSolid` w Emotion + design tokens, demo Vite i Storybook).

Szybkie uruchomienie:

1. Zainstaluj zależności:
```bash
cd /Users/natalia/Praca/MCPtest
npm install
```

2. Uruchom demo (Vite):
```bash
npm run dev
```

3. Uruchom Storybook:
```bash
npm run storybook
```

Jeśli chcesz, mogę też dodać repo Git i wypchnąć na GitHub — potwierdź wtedy, że mogę wykonać `git` w tym katalogu.
